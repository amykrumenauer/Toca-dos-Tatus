# Toca dos Tatus 🐾

**Um site responsivo para adoção de animais e divulgação de atividades de voluntariado.**  
Desenvolvido como projeto pessoal/extensionista, com foco em UX/UI e responsividade.

---

## Descrição do Projeto

https://sitetocadostatus.vercel.app/

O **Toca dos Tatus** é um site que conecta pessoas que querem adotar pets com informações sobre os animais disponíveis e atividades de voluntariado.  
O site possui seções como:

- **Home/Hero**: Apresentação do projeto e chamada para ação.  
- **Sobre**: Informações sobre a organização.  
- **Quero Adotar**: Cards com animais disponíveis para adoção.  
- **Loja**: Produtos relacionados à causa.  
- **Voluntariado**: Formulário e informações para voluntários.  
- **Contato**: Localização, redes sociais e formulário de contato.  

O projeto é **totalmente responsivo**, funcionando em desktop e mobile.

---

## Layout

- Fonte principal: `Julius Sans One` para títulos.  
- Fonte secundária: `Flamenco` para textos e descrições.  
- Paleta de cores:  
  - Azul escuro / #261B1B → fundo da navbar e cards  
  - Verde / #00C29F → títulos  
  - Amarelo / #FFBB00 → destaques e botões  
- Imagens de fundo (patas e pets) com baixa opacidade para efeito visual sem prejudicar a leitura.  

---

## Tecnologias Utilizadas

- HTML5  
- CSS3 (Flexbox, Grid, Media Queries)  
- JavaScript (para menu-hambúrguer interativo)  
- React (via Vite)  
- Git e GitHub para versionamento  

---

## Funcionalidades

- Menu responsivo com **hambúrguer** em mobile.  
- Cards interativos com **efeito hover**.  
- Layout **responsivo** usando `@media queries`.  
- Imagens de fundo posicionadas com `absolute` e `z-index` para profundidade.  
- Links de navegação suave para seções da página.
